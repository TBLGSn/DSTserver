# DSTserver
Don't Starve Together 脚本
## Usage
### 0. 安装 DST
```shell
wget "https://raw.githubusercontent.com/TBLGSn/DSTserver/main/dst.sh" && chmod u+x dst.sh
bash dst.sh
```
### 1. 管理服务器
```shell
bash /home/dst/server_dst/bin/menu.sh
```
